# RepTimer

RepTimer is a simple, mobile-first interval workout timer built with plain HTML, CSS, and JavaScript.

No accounts. No tracking. No distractions.  
Just work, rest, repeat.

## Features

- Work + rest intervals
- Custom number of rounds
- Countdown before starting
- Audio + visual alerts at every interval
- Start / Pause / Resume / Skip / Reset
- Dark mode, bold fitness-focused design
- Runs entirely in the browser

## Live Demo

https://tacofan709.github.io/reptimer/ 

## How It Works

1. Set your:
   - Number of rounds
   - Work time (seconds)
   - Rest time (seconds)
2. Press **Start**
3. Follow the timer until you're done
